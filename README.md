# Portfolio Astro - Modern Creative Developer Portfolio

A high-performance, visually immersive portfolio website built with **Astro**, **React**, and **GSAP**. This project focuses on smooth animations, modular architecture, and a seamless multi-language experience.

---

## 🚀 Technologies

- **Framework:** [Astro](https://astro.build/) (Islands Architecture)
- **UI Library:** [React](https://reactjs.org/)
- **Animation:** [GSAP](https://greensock.com/gsap/) (ScrollTrigger, Custom Ease)
- **Styling:** [SCSS Modules](https://sass-lang.com/), [TailwindCSS](https://tailwindcss.com/)
- **Type Safety:** [TypeScript](https://www.typescriptlang.org/)
- **Content:** [Astro Content Collections](https://docs.astro.build/en/guides/content-collections/)
- **Internationalization:** Custom i18n implementation (EN/VI)

## ✨ Key Features

- **Smooth Interactions:** Advanced GSAP animations for transitions, sticky effects, and element reveals.
- **Islands Architecture:** Optimized performance by hydrating only interactive components.
- **Multilingual Support:** Full English and Vietnamese support using custom routing and translation utilities.
- **Modular Styling:** Combines the utility of Tailwind with the precision and logic of SCSS Modules.
- **Content Driven:** Projects, experiences, and tech stacks are managed via Content Collections for easy updates.

## 🛠️ Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

# Portfolio Astro - Trang Web Portfolio Sáng Tạo Hiện Đại

Một trang web portfolio hiệu năng cao, trải nghiệm thị giác ấn tượng được xây dựng bằng **Astro**, **React** và **GSAP**. Dự án tập trung vào các hiệu ứng chuyển động mượt mà, kiến trúc mô-đun và trải nghiệm đa ngôn ngữ liền mạch.

## 🚀 Công Nghệ Sử Dụng

- **Framework:** [Astro](https://astro.build/) (Kiến trúc Islands)
- **Thư viện UI:** [React](https://reactjs.org/)
- **Hiệu ứng:** [GSAP](https://greensock.com/gsap/) (ScrollTrigger, Custom Ease)
- **Styling:** [SCSS Modules](https://sass-lang.com/), [TailwindCSS](https://tailwindcss.com/)
- **Type Safety:** [TypeScript](https://www.typescriptlang.org/)
- **Quản lý nội dung:** [Astro Content Collections](https://docs.astro.build/en/guides/content-collections/)
- **Đa ngôn ngữ:** Hệ thống i18n tùy chỉnh (Tiếng Anh / Tiếng Việt)

## ✨ Tính Năng Nổi Bật

- **Tương tác mượt mà:** Sử dụng GSAP nâng cao cho các hiệu ứng chuyển cảnh, thẻ dính (sticky cards) và xuất hiện thành phần.
- **Kiến trúc Islands:** Tối ưu hiệu năng bằng cách chỉ kích hoạt JavaScript cho các thành phần cần tương tác.
- **Hỗ trợ Đa ngôn ngữ:** Hỗ trợ đầy đủ Tiếng Anh và Tiếng Việt thông qua hệ thống routing và tiện ích dịch thuật tùy chỉnh.
- **Styling linh hoạt:** Kết hợp tính tiện dụng của Tailwind với logic và độ chính xác của SCSS Modules.
- **Quản lý dữ liệu tập trung:** Các dự án, kinh nghiệm và kỹ năng được quản lý qua Content Collections, giúp việc cập nhật nội dung dễ dàng hơn.

# Chạy môi trường phát triển
npm run dev

# Xây dựng dự án cho production
